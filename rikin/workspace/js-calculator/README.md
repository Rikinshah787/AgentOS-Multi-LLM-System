# JS Calculator (Node.js)

A simple calculator implemented in JavaScript.

## Features
- Basic arithmetic: `add`, `subtract`, `multiply`, `divide`
- CLI interface
- Error handling (e.g., division by zero)

## Installation
```bash
cd js-calculator
npm install
```

## Usage
```bash
npm start
```
Then enter commands like:
```
> add 5 3
Result: 8
> divide 10 2
Result: 5
> exit
```

## Testing
```bash
npm test
```
